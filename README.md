# Květiny pod zámkem

Jednostránková webová prezentace květinářství **Květiny pod zámkem** v Kamenici nad Lipou.

## Co tu je

- `index.html` — kompletní stránka (HTML + CSS + JS v jednom souboru)
- `assets/bouquet.mp4` — scroll-scrubbed hero video kytice (8s, generováno na Hailuo MiniMax / Veo 3.1)

## Lokální spuštění

```bash
python3 -m http.server 8765
```

Otevři http://localhost:8765

## Deploy

Hostováno přes **GitHub Pages** — každý push na `main` se automaticky publikuje.
